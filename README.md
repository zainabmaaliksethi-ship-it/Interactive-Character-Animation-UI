# Interactive-Character-Animation-UI
This interactive character animation UI project is inspired by Arcane and focuses on delivering a smooth, cinematic browsing experience. Each character is presented on a central glossy card placed over a bold, atmospheric background. The card contains key details such as the character’s name, traits, and quotes, designed with a clean and modern visual style.

The interaction is highly fluid: hovering over the card triggers a responsive animation that enhances focus, while clicking transitions seamlessly to the next character. On mouse leave, the card subtly shifts toward the corner, revealing additional more information about the character in a soft, elegant motion. Moving the mouse back restores the main character view with a smooth return animation.

The entire experience is built around transitions, depth, and visual storytelling, creating a premium, immersive interface that feels dynamic, expressive, and alive.
