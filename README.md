# Interactive-Character-Animation-UI
This interactive character animation UI, inspired by *Arcane*, delivers a cinematic browsing experience through smooth transitions and visual storytelling. Each character appears on a glossy central card with key details like name, traits, and quotes, set against a bold atmospheric background.

Hovering enhances focus with responsive animations, clicking transitions seamlessly between characters, and moving the cursor away reveals additional character information through elegant motion. The result is a dynamic, immersive interface that feels polished, expressive, and alive.
