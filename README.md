# Interactive-Character-Animation-UI

This interactive Arcane-inspired character UI is built around clear, user-driven navigation and cinematic transitions. Each character is displayed on a glossy central card over a bold, atmospheric background, highlighting their traits, quotes, and identity in a refined visual style.

Users can navigate between characters using forward and back buttons, with smooth animated transitions between cards. Clicking a card opens a detailed information view, while a back action returns to the original introduction state with seamless motion.

The design replaces implicit interactions with explicit controls, creating a polished and intuitive experience focused on depth, fluid animation, and cinematic storytelling.
